#### Data repository accompanying the paper "Programming with Chebfun. Case study: Richards equation" by Nicolae Suciu (https://arxiv.org/abs/2606.18118)

##### The repository contains Matlab codes with Chebfun extension organizsed in the following folders:

- 'Linearization' - linearization methods in function spaces and applications to steady-state infiltration in soil

- 'Chebfun_FD' - coupled Chebfun-FD L-schemes to solve non-steady problems for Richards equation in one and two spatial dimensions

- 'Accuracy' - estimates of accuracy orders for FD L-schemes with Chebfun2 and Chebfun3
